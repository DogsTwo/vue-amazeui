# accordion
---
>风琴式菜单

#prop

##inset
>type : Boolean
>>是否添加边距。默认为全宽填满容器，设置 inset 属性以后添加 15px 的边距

##behavior
>type : Boolean
>>是否手风琴式,默认手风琴

##noAnimated
>type : Boolean
>>是否关闭动画

#accordionItem
---
>风琴式菜单子类

#prop

##title
>type : String
>>手风琴标题

##active
>type : Boolean
>>是否展开当前的手风琴
